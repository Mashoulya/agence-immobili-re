# Projet scolaire - Qualité de code Front-End

## Objectif 
Analyser, tester et corriger la qualité du code HTML, CSS et JavaScript d'une copie d'un site.

## Technologies
- HTML
- CSS
- JavaScript
- Git /GitHub

## Outils de qualité de code
Pour assurer le code propre et conforme aux standards, les outils suivants peuvent être utilisés :
- **HTMLHint** - vérification des erreurs et bonnes pratiques HTML
- **Stylelint** - analyse et correction du CSS
- **ESLint** - détection d'erreurs et amélioration du JavaScript

## Étapes à réaliser
1. Cloner le projet original.  
2. Créer votre propre dépôt GitHub et y pousser le projet.  
3. Analyser les fichiers HTML, CSS et JS avec les outils de qualité.  
4. Corriger les erreurs détectées et améliorer le code si nécessaire.  
5. Créer des branches de test pour vos modifications (`test-html`, `test-css`, `test-js`).  
6. Fusionner la version finale dans la branche `main`.  
7. Vérifier que le projet fonctionne correctement et reste identique à l’original.
